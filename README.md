#  SQL + Pandas Assignment

##  Overview
This assignment focuses on understanding and applying **SQL queries and Pandas operations** for data analysis.  
It covers basic to advanced concepts used in real-world data handling and database management.

---

##  Files Included
- `queries.sql` → Contains all SQL queries (basic to advanced)  
- `SQL_Assignment_Answers.docx` → Detailed answers with SQL + Pandas explanations  
- `sql_pandas_comparison.ipynb` → Notebook showing SQL vs Pandas operations  

---

##  What This Assignment Covers

###  1. Basic SQL Queries
- SELECT statements  
- Filtering using WHERE  
- Working with conditions (>, BETWEEN, etc.)

---

###  2. JOIN Operations
- INNER JOIN  
- LEFT JOIN  
- Combining multiple tables  
- Real-world use: linking employees with departments  

---

###  3. Aggregation Functions
- COUNT, SUM, AVG, MAX  
- GROUP BY and HAVING  
- Understanding data summaries  

---

###  4. Advanced SQL Concepts
- Window Functions (RANK, LAG, Running Total)  
- CTE (Common Table Expressions)  
- Correlated Subqueries  

---

###  5. SQL vs Pandas
- Same operations implemented in both SQL and Python (Pandas)  
- Helps understand how database queries translate into Python code  

---

###  6. Query Optimization
- Use of EXPLAIN  
- Understanding performance (Seq Scan vs Index Scan)  

---

###  7. Real-World Scenarios
- Employee and department analysis  
- Project budget comparison  
- Finding insights using joins and aggregations  

---

##  Key Learning Outcomes
- Strong understanding of SQL queries  
- Ability to perform data analysis using Pandas  
- Knowledge of real-world database operations  
- Understanding of query performance and optimization  

---

## ✅ Conclusion
This assignment helps in building a strong foundation in **SQL and data analysis**, which is essential for roles like **Data Analyst, Data Engineer, and Python Developer**.
